# Projet Odoo – Gestion des tâches simples

## Description
Ce projet consiste à développer un module Odoo permettant la gestion des tâches simples sans workflow complexe.  
Il permet de créer, consulter, modifier et supprimer des tâches à travers une interface intégrée à Odoo.

## Nom du module
gestion_taches

## Fonctionnalités
- Création de tâches
- Modification des tâches
- Suppression des tâches
- Affichage des tâches dans des vues Odoo
- Gestion des droits d’accès via les règles de sécurité

## Technologies utilisées
- Odoo
- Python
- XML
- Docker
